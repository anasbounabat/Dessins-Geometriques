What is Genuary

Genuary is a month-long generative art challenge held every January. Each day, a prompt is published — a concept, a constraint, a technique — and artists worldwide respond with code.

The goal isn't perfection. It's daily practice, creative limitation, and the discipline of finishing something every single day for a month.

Genuary started in 2021 and has grown into one of the largest creative coding events globally.

This Implementation

Every day is a single self-contained HTML file. No build tools, no frameworks, no bundlers. Just a browser, a text editor, and mathematics.

All 31 sketches use the native HTML5 Canvas 2D API — a deliberate choice to understand the fundamentals deeply rather than abstracting them away.

The result: 31 files, each a complete experiment, immediately runnable in any browser.

Why No Libraries

Using p5.js or Three.js would have been faster. But the point of Genuary is to learn and push yourself, not to ship quickly.

Writing raw canvas code forces you to understand transforms, the rendering loop, coordinate systems, and the actual math behind every visual effect.

The constraint makes you better.

Generative Art

Generative art is art produced (at least in part) by an autonomous system. You write rules and algorithms; the system produces the image.

Key figures: Sol LeWitt (instruction-based art), Vera Molnár (algorithmic drawing), Casey Reas (Processing, the language that started it all).

The art is in the system design, not the execution.
